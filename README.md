# subway_21st
이름하야 21세기 지하철 모웹/앱 프로젝트

## 기능
* 경로 검색
   * 검색 조건
      * 출발역/도착역 선택 (필수)
         * 기술적 문제로 텍스트 입력 또는 선택만 가능
   * 출발/도착 시간 선택
      * 평일, 토요일, 공휴일
      * 없을 시 현재 기준
   * 일단, 최단 거리 기준으로 검색
* 검색 결과 제공
   * 열차 출발/도착 시간
   * 첫차/막차 시간
   * 직전/직후 열차 시간
   * 환승 소요 시간
   * 개찰구 내 화장실 정보
   * 엘리베이터, 에스컬레이터 정보
   * 탑승구 추천
      * 빠른 환승/출구 탑승구 번호
      * 회전율 높은 탑승구 번호
   * 시간표 정보
   * 현재 열차 위치 정보(시간으로)
